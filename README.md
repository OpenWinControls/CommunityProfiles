# Community repo for GPD dual-mode controller mapping profiles

A collection of mapping profiles you can easily import or download from [OpenWinControls](https://github.com/OpenWinControls/OpenWinControls).

## Upload your mapping profiles

Create your mapping in OpenWinControls and export it.

You can edit the exported **yaml** file to add a few optional keys on top to describe your mapping:

**author**: your name

**game**: game title

**desc**: additional info about this mapping, like game settings to change, if needed
